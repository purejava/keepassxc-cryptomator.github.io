# Develop
 Install the dependencies with [Bundler](http://bundler.io/):

 ~~~bash
 $ bundle install
 ~~~

 Run `jekyll` commands through Bundler to ensure you're using the right versions:

 ~~~bash
 $ bundle exec jekyll serve
 ~~~
